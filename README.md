# 기능
- Spring Security와 JWT Token을 이용한 사용자 인증
- 회원가입, 로그인
- 게시글 CRUD
- 게시글 좋아요
- 댓글/대댓글 CRUD
- AWS S3 업로드
  - 프로필 이미지 파일 업로드
  - 게시글 음성 파일 업로드

# 개발환경
- Intellij IDEA Ultimate
- Spring Boot 2.6.10
- Java 11
- Gradle 7.5
